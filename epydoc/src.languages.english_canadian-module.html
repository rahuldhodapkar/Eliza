Module english_canadian

source code

CPSC 599 module: src.languages.english_canadian

Purpose

Contains a collection of rules to apply to phonemes.

Language

This file applies to Canadian English.

Usage

All functions declared in this module for external iteration must have the following input signature:

  • ipa_character (unicode) - The character, representative of a phoneme, being processed.
  • preceding_phonemes (sequence) - A collection of all phonemes, in order, that precede the current IPA character in the current word.
  • following_phonemes (sequence) - A collection of all phonemes, in order, that follow the current IPA character in the current word.
  • word_position (int) - The current word's position in its sentence, indexed from 1.
  • remaining_words (int) - The number of words remaining before the end of the sentence is reached, not including the current word.
  • previous_words (sequence) - A collection of all words that have been previously synthesized.
  • following_words (sequence) - A collection of all words that have yet to be synthesized.
  • sentence_position (int) - The current sentence's position in its paragraph, indexed from 1.
  • remaining_sentences (int) - The number of sentences remaining before the end of the paragraph is reached, not including the current sentence.
  • is_quoted (bool) - True if the current word is part of a quoted body.
  • is_emphasized (bool) - True if the current word is part of an emphasized body.
  • is_content (bool) - True if the current word was marked as a content word.
  • is_question (bool) - True if the current sentence ends with a question mark.
  • is_exclamation (bool) - True if the current sentence ends with an exclamation mark.
  • previous_phoneme_parameters (list) - A collection of all parameters that appear as part of this phoneme, prior to the parameter-set currently being manipulated.
  • remaining_phoneme_parameter_count (int) - The number of parameter-sets yet to be processed as part of this phoneme.
  • previous_sound_parameters (list) - A list of all preceding parameter-sets introduced prior to the current paramter-set by language rules.
  • following_sound_parameters (list) - A list of all preceding parameter-sets introduced after to the current paramter-set by language rules.
  • parameters (list(33)) - A collection of parameters associated with the sound currently being procesed.

Additionally, they must have the following return format:

  • tuple(3) - A list of parameter-sets that precede this sound, a list of parameter-sets that follow this sound, and an f0 multiplier.

All functions may modify the input parameter-set, parameters.

To enable use of a function you have defined, you must add a reference to the RULE_FUNCTIONS tuple, found at the end of this file.
