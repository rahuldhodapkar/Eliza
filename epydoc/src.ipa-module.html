Module ipa

source code

CPSC 599 module: src.ipa

Purpose

Provides IPA-to-parameter lookup functions.

Limitations

At present, only the following IPA symbols are supported:

  • Consonants:
    • mnŋpbtdɾkgfvθðszʃʒhʔɹjlwʍ
  • Vowels:
    • ieɛæaIəʊuoʌɔ

reduceIPAClusters(token)

source code 

Returns the input word as a collection of IPA characters, condensing multi-character symbols into a single string, and leaving extension syntax unchanged.

Parameters:
  • token (unicode) - The word to be reduced.
Returns: list
A list of all IPA characters in the input token, adjusted to handle multi-character symbols, and containing all extension syntax.
